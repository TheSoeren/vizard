## Setup

1. Download lexbor by following this [guide](https://lexbor.com/download/).

of the documentation with the instructions in this [github issue](https://github.com/lexbor/lexbor/issues/217).

2. Download the current Qt6 dev tools

`sudo apt install qt6-base-dev`
